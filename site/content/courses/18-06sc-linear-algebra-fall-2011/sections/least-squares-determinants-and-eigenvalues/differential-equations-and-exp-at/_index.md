Session Overview
----------------

| ![Figure excerpted from 'Introduction to Linear Algebra' by G.S. Strang](https://open-learning-course-data-ci.s3.amazonaws.com/18-06sc-linear-algebra-fall-2011/38c6002bffe782a07c0377b3686b0c41_2_10.jpg) | {{< br >}}{{< br >}}We can copy Taylor's series for _ex_ to define _eAt_ for a matrix _A_. If _A_ is diagonalizable, we can use Λ to find the exact value of _eAt_. This allows us to solve systems of differential equations _d_u / _dt_ = _A_u the same way we solved equations like _dy / dt = ky_.{{< br >}}{{< br >}}
