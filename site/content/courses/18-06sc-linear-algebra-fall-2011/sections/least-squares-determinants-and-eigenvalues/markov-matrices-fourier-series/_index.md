Session Overview
----------------

| ![Figure excerpted from 'Introduction to Linear Algebra' by G.S. Strang](https://open-learning-course-data-ci.s3.amazonaws.com/18-06sc-linear-algebra-fall-2011/5264c401e6567924449455218032bd8e_2_11.jpg) | {{< br >}}{{< br >}}Like differential equations, Markov matrices describe changes over time. Once again, the eigenvalues and eigenvectors describe the long term behavior of the system. In this session we also learn about Fourier series, which describe periodic functions as points in an infinite dimensional vector space.{{< br >}}{{< br >}}
