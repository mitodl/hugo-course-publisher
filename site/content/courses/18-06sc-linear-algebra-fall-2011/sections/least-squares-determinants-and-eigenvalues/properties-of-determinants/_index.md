Session Overview
----------------

| ![Figure excerpted from 'Introduction to Linear Algebra' by G.S. Strang](https://open-learning-course-data-ci.s3.amazonaws.com/18-06sc-linear-algebra-fall-2011/a26793429534b75a1c47bda84e444859_2_5.jpg) | {{< br >}}{{< br >}}The determinant of a matrix is a single number which encodes a lot of information about the matrix. Three simple properties completely describe the determinant. In this lecture we also list seven more properties like det_AB_ = (det_A_)(det_B_) that can be derived from the first three.{{< br >}}{{< br >}}
