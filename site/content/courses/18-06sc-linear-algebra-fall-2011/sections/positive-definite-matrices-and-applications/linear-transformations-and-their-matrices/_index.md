Session Overview
----------------

| ![Figure excerpted from 'Introduction to Linear Algebra' by G.S. Strang](https://open-learning-course-data-ci.s3.amazonaws.com/18-06sc-linear-algebra-fall-2011/94cdc4af9e1094bf8301b6eca90fa986_3_6.jpg) | {{< br >}}{{< br >}}When we multiply a matrix by an input vector we get an output vector, often in a new space.
