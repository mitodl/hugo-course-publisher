Session Overview
----------------

| ![Figure excerpted from 'Introduction to Linear Algebra' by G.S. Strang](https://open-learning-course-data-ci.s3.amazonaws.com/18-06sc-linear-algebra-fall-2011/f088e466f64ba55ee2731afc8e4760e2_1_10.jpg) | {{< br >}}{{< br >}}For some vectors b the equation _A_x = b has solutions and for others it does not. Some vectors _x_ are solutions to the equation _A_x = 0 and some are not. To understand these equations we study the column space, nullspace, row space and left nullspace of the matrix _A_.{{< br >}}{{< br >}}
